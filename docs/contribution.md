# K2G + FPS-ID + CW の学術的意義と方法論的位置づけ

## 【はじめに】

本稿は、古典語テキスト（『伊勢物語』『土佐日記』等）を対象とした統合的解析パイプライン——すなわち、

- **K2G**（古語形態素解析）
- **FPS-ID**（意味ID生成）
- **CW**（共起構造の重みづけ）

の各モジュールが、現代の言語学・文学研究・言語教育においてどのような学術的価値を持ちうるかを明確にし、単なるツール構築にとどまらない理論的貢献を明示するものである。

---

## 【1. 背景：従来技術の限界】

### 1.1 MeCab/Juman++ による古語解析の限界

- 連接規則と現代語品詞体系に依存 → 語構造の逸脱・文法的断絶（例：「いますがり」「かくなりぬ」）を正しく扱えない
- 詩的省略・接尾の分離・補助構造などがブラックボックス化

### 1.2 分散意味表現（word2vec/fastText）の限界

- 古典語コーパスでは訓練量が不十分（数十万語規模）
- 語義の階層的な可視化や抽象度制御が不可能
- 語の意味的まとまりが「単なるベクトル空間の距離」に還元されすぎる

---

## 【2. K2G の学術的意義】

### 2.1 文脈構造に基づく形態素分割

- Leipzig Glossing Rules 準拠の gloss を基盤とすることで、形態素の意味機能と構文的統語性を同時に表現可能

### 2.2 接続規則を排除した設計思想

- 「語り」や「逸脱構文」を許容する構造的柔軟性
- 和歌や説話など、創造的連接の観察を阻害しない

### 2.3 教育応用と文法理解

- 助動詞や補助構造（PST.REP, SUPP, HON など）の形式化により、古語教育や翻訳支援にも応用可能

---

## 【3. FPS-ID の理論的貢献】

### 3.1 意味の連続性と桁構造による抽象度制御

- 上位桁（例："23"）＝語義大分類、下位桁（"23.6147"）＝文脈的語感という多層的構造

### 3.2 頻度ベースによる意味空間構築

- データ量に乏しい古典語でも「共起ベースの意味分布」は再現可能
- 意味の座標化・数値化 → 比較・分類・変遷研究に接続

### 3.3 固定カテゴリ辞書からの脱却

- BGH（分類語彙表）などの静的分類に代わり、文脈から動的に形成される意味クラスタの可能性

---

## 【4. CW の分析的有効性】

### 4.1 意味クラスタ単位の重みづけ

- 共起ベースで語群を形成し、それらの関係強度を `ctf × √(idf)` で定量化

### 4.2 視覚化による解釈可能性

- Graphviz等による意味ネットワークの描画 → 解釈・発見・仮説生成の支援ツールとして機能

### 4.3 和歌的連想と語りの連鎖モデルへの応用

- 伝統的和歌に見られる「意味の取り合わせ」「語感の跳躍」の構造を定量的に可視化

---

## 【5. 方法論的新規性と比較優位】

| 項目       | 本プロジェクト        | 従来技術               |
| ---------- | --------------------- | ---------------------- |
| 品詞構造   | gloss に基づく文脈型  | MeCab等の固定タグ体系  |
| 意味ID     | PCAによる連続階層型   | 固定分類（例：BGH）    |
| 構文制御   | 非連接規則型          | 接続表に基づく固定遷移 |
| 語義可視化 | 桁数制御 + グラフ描画 | 無視される or 手動記述 |

---

## 【6. 応用分野と展望】

- 文献学：語義変遷・写本間の語用差
- 詩学：和歌・連歌における語の意味連鎖の構造化
- 教育：古語文法の語義機能別可視化教材
- 翻訳：意味ごとの訳語対応、語調・敬意・語り機能の支援

---

## 【結論】

K2G・FPS-ID・CW からなる解析系は、単なる技術ではなく、

- 古語テキストの内部構造を言語学的・詩学的に「見る」ための方法論
- 創造的言語表現と構文逸脱を“正規の構造”として扱う理論枠
- 語り・語用論・語義構造に対する定量的視座

を提供する。

このパイプラインは、古語研究を**構造の発見と連関の観察**に変えるための、堅実かつ斬新な方法論といえる。
