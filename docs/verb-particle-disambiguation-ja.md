
# 動詞に基づく格判定ルール一覧

## 概要

本資料は、日本語の動詞が要求する格要素を体系的に整理し、助詞「に」の意味判定を支援する目的で構築する。主に k2g (Koten to Gloss) や gloss-lint システムの格同定モジュールに活用されることを想定している。

---

## 格ラベル定義（抜粋）

| ラベル  | 名称      | 説明例        |
| ---- | ------- | ---------- |
| SUBJ | 主語格     | 「彼が」「私が」   |
| OBJ  | 目的格     | 「本を読む」     |
| DAT  | 与格      | 「彼に言う」     |
| LOC  | 場所格     | 「机に置く」     |
| ALL  | 到達格/方向格 | 「東京に行く」    |
| TIME | 時間格     | 「九月に生まれる」  |
| PURP | 目的格     | 「遊びに行く」    |
| CAUS | 原因格     | 「病気に倒れる」   |
| BEN  | 恩恵格     | 「親に買ってあげる」 |
| ABL  | 起点格     | 「家から出る」    |
| INST | 手段格     | 「筆で書く」     |
| QUOT | 引用格     | 「と言った」     |
| COMP | 比較格     | 「誰よりも速い」   |

## 動詞ごとの格要求一覧（代表例）

### A. 一般的な動詞分類による要求格

#### 1. 移動動詞（行く・来る・帰る・通う など）

| 動詞 | 格要素 | 助詞 | gloss     |
| -- | --- | -- | --------- |
| 行く | 目的地 | に  | ALL       |
| 来る | 目的地 | に  | ALL       |
| 帰る | 出発地 | から | ABL       |
| 通う | 通過点 | に  | LOC / ALL |

#### 2. 認知・伝達動詞（言う・教える・知らせる・話す）

| 動詞  | 格要素 | 助詞 | gloss |
| --- | --- | -- | ----- |
| 言う  | 相手  | に  | DAT   |
| 教える | 相手  | に  | DAT   |
| 話す  | 相手  | に  | DAT   |
| 話す  | 話題  | を  | OBJ   |

#### 3. 感覚・心理動詞（見える・聞こえる・好き・会う など）

| 動詞   | 格要素 | 助詞 | gloss |
| ---- | --- | -- | ----- |
| 好きだ  | 対象  | が  | SUBJ  |
| 会う   | 対象  | に  | DAT   |
| 驚く   | 原因  | に  | CAUS  |
| 感動する | 原因  | に  | CAUS  |

#### 4. 行為・操作動詞（使う・書く・読む・食べる）

| 動詞  | 格要素 | 助詞 | gloss |
| --- | --- | -- | ----- |
| 使う  | 道具  | を  | OBJ   |
| 書く  | 道具  | で  | INST  |
| 書く  | 媒体  | に  | LOC   |
| 食べる | 対象  | を  | OBJ   |

## 実装に向けた設計指針

* 動詞語彙に対して、

  * 基本義での主要格パターンを定義
  * 意味拡張（メタファー的用法）は別途オプションでマーク
* 格判定では、

  * 動詞に応じて「に」を DAT / LOC / ALL / CAUS / PURP 等に分岐
  * 必要に応じて「複数候補」付きでマーク

---

## 今後の予定

* よく使われる動詞100語の格要求辞書化（例：「与える」「載せる」「預ける」など）
* 文脈からの disambiguation モデル設計（動詞 + 名詞 + 活用形）
* 古語動詞に対応した格要素の検討（例：「まかる」「まゐる」）

---

この一覧は拡張可能です。必要に応じて新しい動詞や構文タイプを追加していくことで、精度を高められます。
